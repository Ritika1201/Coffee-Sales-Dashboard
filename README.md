# ☕ Coffee Sales Dashboard
Interactive Coffee Sales Dashboard built using Microsoft Excel and Tableau.

##  Project Overview
This project presents an interactive Coffee Sales Dashboard created using Microsoft Excel and Tableau. The data was cleaned in Excel and visualized in Tableau to uncover sales trends, customer behavior, and product performance.

## Tools Used
- Microsoft Excel
- Tableau

## 📂 Project Workflow
1. Collected raw coffee sales data.
2. Cleaned and prepared the dataset in Microsoft Excel.
3. Imported the cleaned data into Tableau.
4. Created KPIs and interactive visualizations.
5. Generated business insights from the dashboard.

## 📊 Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Payment method KPI
- Weekly sales
- Top 10 Customers
- Monthly Sales Trend
- Category-wise Sales Analysis
- Peak Ordering time

##  Key Insights
- Identified the best-selling coffee products.
- Analyzed monthly sales trends.
- Identified top customers based on orders.
- Compared sales across different product categories.
